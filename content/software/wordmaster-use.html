After Effects英文界面学习助手 - 用户手册

产品概述

AE Word Master 是一款专为After Effects（AE）用户设计的桌面应用软件，旨在帮助英文基础薄弱的用户快速熟悉和掌握After Effects英文版界面中的专业术语。通过系统化的学习和练习，用户能够无障碍使用英文版After Effects，提升工作效率和专业水平。

环境要求

1. 操作系统

  • 推荐系统： Windows 10 或 Windows 11（64位）
  • 部分兼容系统： Windows 7 SP1（64位）
  • 不支持的系统： macOS、iOS、Android、Linux 及 Windows 32位系统

2. 硬件配置

  • 内存： 4GB 及以上（建议 8GB 以获得更好体验）
  • 处理器： Intel Core i3 或更高性能处理器
  • 硬盘空间： 1GB 可用空间
  • 显示器： 1920×1080 分辨率

安装指南

Windows系统安装

  1. 下载解压安装文件 AE Word Master Setup 2.0.0.exe
  2. 双击运行安装程序
  3. 在安装向导中，选择您希望的安装模式：

  • ✅为所有用户安装：需要管理员权限，将应用安装在 C:\Program Files\
  • ✅仅为当前用户安装：无需管理员权限，将应用安装在您的个人目录下

  1. 选择安装路径（或使用默认路径）
  2. 点击“安装”并等待完成，安装完成后，桌面和开始菜单将创建快捷方式

分区与模块核心功能详解

应用界面采用经典的三栏式布局，顶部为功能操作区，左侧为术语导航区，中央为学习浏览区。整体设计注重高效浏览与深度学习。

1. 顶部操作栏（Header）

位于窗口最上方，包含品牌标识和核心操作按钮。

  • 品牌标识区：左侧显示“AE英文界面学习助手”。
  • 功能按钮区：中部排列的快捷按钮是核心操作入口。
  • “＋ 添加单词”：点击顶部 “＋ 添加单词”，在弹出的窗口中填写英文、中文和详细描述后保存，该术语会出现在 “个人添加” 分类中。
  • “📥 导入”：点击 “📥 导入”，可选择导入标准JSON词库文件(需要了解json结构)，或以“英文=中文”格式的文本笔记。
  • “📤 导出个人笔记”：点击“📤 导出个人笔记” 可以导出自行添加的单词以备份自定义内容。
  • “💎 导出全量 (VIP)”：点击“💎 导出全量 (VIP)” 用于获取包含官方数据的完整包。（需VIP激活）。

​ VIP与激活系统

  • 点击 “💎导出全量 (VIP)” 按钮，会触发VIP模态框。界面会显示一个付款二维码和唯一的机器码
  • 用户需按下方的价格扫码支付，并复制机器码，做好记录。
  • 将机器码发给开发者，将获得一个12位激活码，输入即可解锁全量导出功能。
  • “❤️ 支持开发者”：查看赞助二维码与支持信息。
  • 如果本软件对你有用，可以扫码支持开发者，你的支持是我们持续更新的动力。
  • 学习进度区：右侧显示“已掌握 X / Y”，实时反馈学习成果。

2. 左侧导航栏（Sidebar）

位于窗口左侧，是管理和筛选所有学习内容的核心面板。

  • 搜索框：支持输入中英文关键词，实时筛选术语列表。
  • 分类筛选下拉菜单：用于切换不同视图。
  • “📁 所有分类”：浏览全部术语。
  • “✅ 已掌握”：仅查看已被标记掌握的术语。
  • “⭐ 我的收藏”：查看所有收藏的术语。
  • “🚩 难点标记”：查看所有标记为“难点”的术语。
  • “👤 个人添加”：查看所有用户自定义添加的术语。
  • 术语列表：按所选分类动态加载，以列表形式展示所有术语。点击列表中的任意术语，中央详情区将立即显示其完整内容。

3. 中央学习浏览区（Main Viewer）

占据界面主要区域，用于详细展示和交互学习。包含两种视图：

  • 欢迎视图：应用启动时默认显示，仅展示品牌标识 szsoar。
  • 术语详情视图：点击左侧列表任意术语后激活，是学习的主要界面。

1. 术语元信息栏：顶部显示该术语的所属分类标签、“官方原版”标识，以及其收藏、难点状态徽章。

2. 核心术语展示区：

  • 英文术语：以大号字体突出显示（如 Keyframe）。
  • 音标与发音：显示音标，并提供 “🔊” 发音按钮。
  • 中文释义：清晰展示对应中文翻译。

3. 状态操作栏：

  • “✅ 标记掌握” / “⭐ 收藏” / “🚩 难点”：三个核心按钮，用于更新该术语的个人学习状态。
  • “📝 编辑” / “🗑️ 删除”：仅对用户个人添加的术语可见，用于管理自定义内容。
  • “自动隐藏已学内容”复选框：勾选后，左侧列表将自动隐藏已掌握的术语。

4. 功能详细说明面板：

  • 标题注明“功能详细说明 (OFFLINE DATA)”。
  • 下方大段文字区域展示该术语在After Effects中的详细功能描述、应用场景及操作要点。

5. 导航控制栏：位于面板底部。

  • “⬅ 上一个” / “下一个 ➡”：在当前筛选出的列表中顺序切换。
  • “🎲 随机测试”：随机跳转到列表中的一个术语，用于复习和自测。

4. 页脚（Footer）

位于窗口底部，仅显示版权信息：© 2026 szsoar. All Rights Reserved.

常见问题解答

Q1：软件支持离线使用吗？

A：完全支持离线使用。所有功能均不需要网络连接，确保您在任何环境下都能学习。

Q2：我能添加多少自定义术语？

A：理论上没有限制，但建议每个术语提供完整信息以获得最佳学习效果。

Q3：软件会收集我的数据吗？

A：不会。所有数据都存储在本地，我们不会收集任何用户数据。请定期备份重要数据。

Q4：如何将学习数据迁移到新电脑？

A：使用"导出个人笔记"功能备份，然后在新电脑上安装软件后使用"导入"功能恢复。

Q5：软件支持多用户吗？

A：当前版本为单用户设计。如需多用户支持，请为每个用户单独安装并管理数据。

Q6：发音功能需要额外安装吗？

A：不需要。软件使用系统自带的语音合成功能，Windows系统可直接使用。

Q7：软件支持哪些After Effects版本？

A：术语覆盖After Effects CC 2014到最新版本，核心术语适用于所有版本，并在后期持续更新

Q8: 学习进度会同步吗？

当前版本为单机应用，所有学习数据保存在本地。未来版本可能考虑加入云同步功能。

故障排除

问题1：软件无法启动

解决方案：

  1. 确保系统满足最低要求
  2. 以管理员身份运行（Windows）
  3. 重新安装最新版本
  4. 检查防病毒软件是否误拦截

问题2：发音功能无法使用

解决方案：

  1. 检查系统语音设置
  2. 确保安装了语音包

问题3：数据丢失

解决方案：

  1. 检查是否有最近的备份文件
  2. 如无法恢复，可重装软件以恢复基础术语

问题4：界面显示异常

解决方案：

  1. 重启软件
  2. 检查显示分辨率设置
  3. 更新显卡驱动
  4. 调整界面缩放

问题5：搜索功能不工作

解决方案：

  1. 确保输入了搜索关键词
  2. 检查当前分类中是否有匹配内容
  3. 尝试重新启动软件
  4. 软件可能正在加载词库有延迟
